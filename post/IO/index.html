何为IO

简单的说,IO即为输入/输出(Input/Output),我的理解是一种数据交互的行为。 因为我们的计算机是基于数字来表达信息，我们的计算机也不是仅仅存储数据(当然存储也是要走磁盘IO的), 更多的是信息的交互,互联网时代,数据/信息的价值也显得尤为重要。有数据，有交互那就自然有IO,简单的看,似乎可以将数据进入内存认为Input,反之数据从内存交付给外部认为是Output。

Java 里的IO

作为一名Java工程师，我将从Java的角度下闲谈一下IO。Java API 给我们提供了两个基础接口InputStream/OutputStream用来操作数据流。
